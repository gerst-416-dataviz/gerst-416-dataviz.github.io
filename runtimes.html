CS416 Data Visualization - "Movies Across the Decades" - William Gerst, Summer 2023
On the left, we now have a scatterplot of runtime (in minutes) versus release year for all movies with at least 5,000 user ratings, again scaled to reflect the number of ratings.

We hope that despite the bias we observed before, we will still be able to make inferences about any trends in movie runtime over the years. Have they gotten longer, shorter, or stayed the same?

Note that the vertical axis is now a logarithmic scale, since the runtimes vary all the way from 45 to nearly 570 minutes long.

Even though we started with a vote threshold of 5,000 votes this time, it's still quite cluttered, since this includes over 16,000 movies.
Made with D3 and D3-Annotations. Uses data from IMDb non-commercial datasets. This is intended to be viewed on desktop with a high-resolution display.
